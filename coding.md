要求：
- 逐行解释核心逻辑

- 给出优化后的版本
- 解释优化点

[在这里粘贴你的代码]
```

错误信息：
[粘贴错误信息]

[粘贴代码]
```

3. 新功能实现
```
你是一位全栈工程师。请使用 Python + FastAPI（或 Next.js）帮我实现以下功能：
需求：...
要求：
- 代码结构清晰，包含注释
- 考虑异常处理和边界情况
- 提供使用示例
```

4. 代码审查（Code Review）
```
请以 Senior Engineer 的身份，对下面这段代码进行全面 Code Review。
重点关注：可读性、性能、安全性、可维护性、设计模式。
给出具体改进建议和重构代码。
```

### 使用建议
- 把具体代码或错误信息替换进去
- 可以指定语言（如 “用 Rust 实现”）
- 对于复杂需求，可以分步让模型先给出架构，再写代码
