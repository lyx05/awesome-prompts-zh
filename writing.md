# ✍️ 写作类 Prompt

1. 正式邮件 / 请假邮件
你现在是一名经验丰富的职场助理。请帮我写一封简短、礼貌且专业的请假邮件。
要求：

语气正式但不生硬
说明请假原因（可模糊处理）
提出具体请假日期
表达愿意配合工作交接
结尾感谢对方

请假原因：身体不适
请假日期：2026年4月15日（周三）一天

2. 营销文案（短视频/朋友圈）
你是一位顶级文案策划师。请为以下产品写一条适合朋友圈/短视频的营销文案。
产品：XXXX
核心卖点：XXXX
目标人群：XXXX
语气要求：温暖、信任、带一点生活感
字数控制在 80-120 字
请给出 3 个不同版本供我选择。

3. 小说/故事写作
你是一位畅销小说作家。请根据以下设定，帮我续写下一段故事（800字左右）。
要求：

保持原有风格和人物性格
增加合理冲突或转折
结尾留下悬念
设定：...

使用建议
- 把具体需求替换成你的实际情况
- 可以添加 “用中文回复” 来确保输出语言
- 多版本对比后选择最好的
